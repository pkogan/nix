Que hace nix?

Es una Aplicación web y movil que le permite a los guardavidas

Caracteristicas

Es una Aplicación web y movil que le permite a los guardavidas

Desarrollada con herramientas Libres

Codigo fuente en repositorio https://github.com/pkogan/nix. Desarrollo sobre plataforma php-apache-mysql. Framework Yii2

Seguridad

Datos guardados en base de datos mysql. Permite el acceso con diferentes niveles de Seguridad

Se puede instalar en servidores propios

Análisis de datos

Permite Análisis de datos histórico con variables georáficas, temporales y características de asistencias

Subir audios e imágenes

Permite adjuntar audios e imágenes asociadas a las asistencia

Características

Se permiten guardar novedades, primeros auxilios, prevenciones y rescates
